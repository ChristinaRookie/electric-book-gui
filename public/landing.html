The Electric Book Manager uses your GitHub account to manage your books. If you don't have one already, go to GitHub and sign up for an account.

On Step 2 choose the free plan and don't tick the box for "Help me set up an organization next."

You'll need to tell GitHub that you grant permission for the Electric Book Manager to update the books that you keep on GitHub. They call this "authorizing an application". You only need to do this once.

Authorize the Electric Book Manager on GitHub

Once you've done that you'll be automatically brought be back here, and see a list of all your books.
